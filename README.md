# Alert Automation Examples

This repo provides example automation scripts for alert related tasks, including:

* [Creation of alerts](./createAlerts/README.md)
* [Moving alerts from one account to another](./migrateAlerts/README.md)
* [Enable/disable of alerts](./muteAlerts)

View each separate repo for additional details around the examples above.
